Furniture Store - Penjualan barang rumah

E-commerce modern yang menjual Furniture atau peralatan/barang barang rumah, seperti meja untuk makan/belajar/ruang tamu, kursi, lemari, dan lain lain.
Menggunakan React dan express, disajikan dengan UI/UX menarik supaya sesuai dengan furniture yang indah, dengan tema SAGE seperti dedaunan hijau, full mulai dari login - home
